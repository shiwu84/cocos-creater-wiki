## Lesson Content

欢迎开启您的 **Linux 之旅**！如果您准备好深入探索强大的 Linux 世界，那么您来对地方了。我叫企鹅皮特（Penguin Pete），我将担任您的向导。首先，让我们简要了解一下 **Linux 的历史**。

### Linux 的前身

要了解 Linux 是如何诞生的，我们必须回到 1969 年，当时贝尔实验室的肯·汤普森（Ken Thompson）和丹尼斯·里奇（Dennis Ritchie）开发了 UNIX 操作系统。后来，它使用 C 语言进行了重写，这使其具有了可移植性，并得到了广泛应用。

![Unix 时间轴](https://file.labex.io/images/ed9c245d-e8be-4287-bf34-67750b042542.jpg)

十多年后，理查德·斯托曼（Richard Stallman）发起了 GNU（“GNU's Not UNIX”的递归缩写）项目。其目标是创建一个完全免费且开源的类 UNIX 操作系统。虽然 GNU 项目生产了许多重要组件，包括 GNU 通用公共许可证（GPL），但其自身的内核（名为 Hurd）未能及时完成。

### 内核的作用

内核是操作系统的核心组件。它充当桥梁，允许硬件与软件进行通信。内核管理系统资源，例如 CPU、内存和外围设备。本质上，内核控制着系统上发生的一切。虽然当时正在开发 BSD 和 MINIX 等其他类 UNIX 系统，但它们都缺乏一个免费且统一的内核。

### Linux 内核的诞生

这把我们带到了 1991 年，当时一位名叫林纳斯·托瓦兹（Linus Torvalds）的芬兰学生开始将开发一个新内核作为个人项目。这个内核，也就是我们现在所知的 Linux 内核，填补了 GNU 操作系统的缺失部分。GNU 工具与 Linux 内核的结合，创造了今天广泛使用的完整开源操作系统。这一里程碑是 **Linux 历史**上的关键时刻。
