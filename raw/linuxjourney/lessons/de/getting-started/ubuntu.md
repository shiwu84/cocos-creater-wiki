## Lesson Content

### Was ist Ubuntu?

Ubuntu ist eine der am weitesten verbreiteten Linux-Distributionen. Es wird von Canonical entwickelt, basiert auf Debian und ist bekannt für sein zugängliches Design, eine große Benutzer-Community sowie umfassende Hardware- und Softwareunterstützung.

Ubuntu ist zu einem gängigen Ausgangspunkt für Menschen geworden, die Linux erlernen möchten, ohne mit einer manuelleren oder fortgeschritteneren Einrichtung zu beginnen. Es wird auf PCs, Entwicklungssystemen, Cloud-Plattformen und Servern eingesetzt, was ihm eine Reichweite verleiht, die nur wenige andere Distributionen erreichen.

### Warum Ubuntu so beliebt ist

Ubuntu ist beliebt, weil es versucht, Linux für den täglichen Gebrauch praktisch zu gestalten. Es bietet einen ausgereiften Installer, eine starke Dokumentation, vorhersehbare Veröffentlichungszyklen und ein großes Ökosystem an Tutorials und Unterstützung durch Dritte. Für viele Benutzer macht diese Kombination Ubuntu zu einer der am einfachsten zu handhabenden Linux-Distributionen.

Ein weiterer Grund für die Sichtbarkeit von Ubuntu ist, dass es in vielen Umgebungen funktioniert. Sie finden es auf Laptops und Desktops, in virtuellen Maschinen, auf Servern und auf Cloud-Plattformen. Diese breite Akzeptanz stärkt seinen Ruf als Linux-Distribution für allgemeine Zwecke.

### Ubuntu und Debian

Ubuntu ist eine Debian-basierte Distribution, was bedeutet, dass es einen Großteil seines Paketverwaltungsmodells und seines Software-Packaging-Ansatzes von Debian erbt. Wenn Sie lernen, wie `apt` unter Ubuntu funktioniert, hilft Ihnen dieses Wissen auch dabei, andere Debian-basierte Systeme zu verstehen.

Gleichzeitig ist Ubuntu nicht nur "Debian mit einem Desktop". Es hat seinen eigenen Veröffentlichungsplan, eigene Standardeinstellungen, ein eigenes Support-Modell und ein eigenes Ökosystem. Wenn Sie es mit anderen Optionen vergleichen möchten, lesen Sie [Wahl einer Linux-Distribution](https://labex.io/de/lesson/choosing-a-linux-distribution) oder erfahren Sie mehr über [Debian](https://labex.io/de/lesson/debian).

### Ubuntu-Veröffentlichungen

Ubuntu verwendet zwei Hauptarten von Veröffentlichungen. Es erscheint alle sechs Monate eine neue Version, und alle zwei Jahre wird eine dieser Versionen zu einer Long Term Support (LTS)-Version. LTS-Versionen werden häufig für Desktops, Workstations und Server gewählt, die eine stabilere Basis benötigen.

Dieses Veröffentlichungsmodell erklärt die Attraktivität von Ubuntu. Benutzer, die eine zuverlässige Basis wünschen, wählen oft LTS, während Benutzer, die neuere Funktionen wünschen, die Zwischenversionen nutzen können, die in einem schnelleren Rhythmus erscheinen.

### Paketverwaltung

Als Debian-basiertes System verwendet Ubuntu das `.deb`-Paketformat und den `apt`-Paketmanager zum Installieren, Aktualisieren und Entfernen von Software. Dies gibt Benutzern Zugriff auf ein sehr großes Software-Ökosystem und einen vertrauten Befehlszeilen-Workflow.

Die Paketverwaltung ist eine der praktischen Stärken von Ubuntu, da sie ausgereifte Debian-Werkzeuge mit einer großen, gut dokumentierten Softwareumgebung kombiniert.

### Desktop- und Servernutzung

Ubuntu wird sowohl auf Desktop- als auch auf Serversystemen eingesetzt. Auf der Desktop-Seite ist es für eine ausgefeilte GNOME-basierte Erfahrung und relativ zugängliche Standardeinstellungen bekannt. Auf der Server-Seite wird es häufig in der Entwicklung, der Web-Infrastruktur und in Cloud-Umgebungen eingesetzt.

Diese Bandbreite macht Ubuntu attraktiv für Benutzer, die eine einzige Linux-Distribution suchen, die vom Lernen auf einem Laptop bis zum Ausführen von Workloads in der Produktion skaliert werden kann.

### Warum Anfänger Ubuntu wählen

Ubuntu wird Anfängern oft empfohlen, da es einfacher zu installieren und bei Problemen zu beheben ist als viele andere Linux-Distributionen. Die große Benutzerbasis bedeutet, dass viele Tutorials, Forenbeiträge und Anleitungen verfügbar sind, wenn etwas schiefgeht.

Für Benutzer, die eine anfängerfreundliche Linux-Distribution suchen, ohne auf langfristige Flexibilität zu verzichten, bleibt Ubuntu einer der sichersten Ausgangspunkte.
