# 纹理贴图

> [!abstract] 摘要
> 纹理贴图（Texture2D）是用于 GPU 采样的图像资源，广泛用于 3D 模型贴图和 UI 精灵渲染，支持多种过滤模式和寻址模式。

## 概述

纹理贴图资源由图像资源（ImageAsset，如 PNG/JPEG）转换而来，通常用于 3D 模型渲染（反射贴图、环境光遮罩等）和 2D UI 渲染。Texture2D 支持邻近/线性过滤（含 Mipmap）以及 Repeat、Clamp-to-edge、Mirrored-repeat 三种寻址模式。Mipmap 可提升远距离渲染性能和减少锯齿，但额外占用约三分之一内存。
