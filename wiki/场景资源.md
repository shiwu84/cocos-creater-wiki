# 场景资源

> [!abstract] 摘要
> 场景资源是 Cocos Creator 中组织游戏内容的中心载体，保存了场景节点树及全局渲染设置，是游戏创作的基础单元。

## 概述

场景（Scene）是呈现给玩家所有游戏内容的载体，场景文件本身作为资源存在，保存了游戏的大部分信息。场景资源的根节点可设置自动释放策略、环境光、阴影、全局雾和天空盒等全局属性。场景切换可通过 `director.loadScene` API 在运行时动态实现。
