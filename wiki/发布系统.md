# 发布系统

> [!abstract] 摘要
> Cocos Creator 的构建发布系统支持将项目一次开发后发布到 Web、小游戏、原生（iOS/Android/Windows/macOS/HarmonyOS）等多个平台。通过构建发布面板配置通用及平台专属选项，支持命令行构建和自定义构建扩展，实现自动化发布流程。

## 核心概念

### 双内核架构
Cocos Creator 基于双内核引擎架构：
- **C++ 内核**：用于原生平台（iOS、Android、Windows、macOS、HarmonyOS）
- **TS 内核**：用于 Web 和小游戏平台

实现一次开发，全平台运行。

### 支持平台

| 类别 | 平台 |
|------|------|
| Web | Web Mobile（H5）、Web Desktop |
| 小游戏 | 微信、抖音、支付宝、淘宝、OPPO、vivo、华为快游戏、荣耀、百度 |
| 原生移动 | iOS、Android、HUAWEI AppGallery Connect |
| 原生桌面 | Windows、macOS |
| 鸿蒙 | HarmonyOS、HarmonyOS Next |
| 其他 | Facebook Instant Games、Google Play、Google Play On Games |

## 关键细节

### 构建发布面板
通过菜单 **项目 -> 构建发布**（快捷键 `Ctrl/Cmd + Shift + B`）打开。面板支持：
- 选择发布平台
- 配置通用及平台专属构建选项
- 导入/导出构建配置（JSON 格式）
- 构建任务管理（构建队列、取消构建、查看日志）

### 通用构建选项

| 选项 | 说明 |
|------|------|
| 发布路径 | 指定构建输出目录（绝对/相对路径） |
| 构建任务名称 | 构建任务的标识名，默认以平台名称 + 递增后缀 |
| 初始场景 | 游戏启动时的首场景 |
| 参与构建场景 | 选择哪些场景参与打包，按需减少包体 |
| Bundles | 选择参与构建的 Asset Bundle（v3.8+） |
| MD5 缓存 | 资源文件名加 MD5 信息，解决 CDN 缓存问题 |
| 主包压缩类型 | 设置主包资源的压缩方式 |
| 配置主包为远程包 | 将主包配置为远程加载 |
| 资源服务器地址 | 指定远程资源服务器 URL |
| 调试模式 | 关闭后进入发布模式（压缩混淆），开启则保留调试信息 |
| Source Maps | 生成 source map 便于调试 |
| 替换插屏 | 自定义启动画面 |
| 引擎模块配置 | 选择 2D/3D 物理后端、WebGL 版本、Wasm/AsmJS 模式等（v3.8.3+） |

### Web 平台特有选项
- **Web Mobile**：游戏视图撑满浏览器窗口
- **Web Desktop**：固定分辨率（默认 1280x960），支持 WebGPU
- 资源服务器地址、设备方向、Polyfills、vConsole

### 微信小游戏特有选项
- AppID（必填，测试用 `wx6ac3f5090a6b99c5`）
- 设备方向、初始场景分包
- 生成开放数据域工程模板
- 分离引擎（引擎插件）
- 高性能模式
- 主包体积限制 4MB，超出需配置远程资源

### 原生平台发布准备
- **Android**：安装 JDK 17、Android Studio 2022.2+、配置 Android SDK/NDK
- **iOS/macOS**：安装 Xcode 14+
- **Windows**：安装 Visual Studio 2019/2022（勾选 C++ 桌面开发和游戏开发）
- 在编辑器偏好设置中配置 SDK/NDK 路径

### 命令行发布
支持通过命令行自动化构建：
```bash
CocosCreator --project path --build "platform=web-desktop;debug=true"
```
支持导出构建配置 JSON、指定 `configPath`、独立发布 Bundle 等功能。退出码 36=成功，32/34=失败。

### 构建扩展
构建流程支持通过扩展注入平台处理逻辑，开发者可：
- 自定义构建模板
- 自定义构建流程（`custom-build-plugin.md`）
- 在扩展管理器中查看内置平台扩展

## 与其他系统的关系

- [[引擎架构]]：双内核架构使跨平台发布成为可能，C++ 内核负责原生平台，TS 内核负责 Web/小游戏
- [[Cocos Creator 概述]]：跨平台发布是 Cocos Creator 的核心特性之一
- [[资源系统]]：Asset Bundle、远程资源加载、分包策略与构建选项紧密相关
- [[场景与节点系统]]：初始场景和参与构建场景决定打包内容

## 注意事项

- 发布路径不允许包含空格、非法字符和中文字符
- 微信小游戏主包不能超过 4MB
- Web 平台默认以 Web Audio 模式加载音频，受浏览器自动播放策略限制
- 原生平台重新构建时仅更新项目资源，不会覆盖已定制的原生工程内容
- 命令行构建需要 GUI 环境（Jenkins 需运行在 agent 模式）
- MD5 缓存启用后，第三方资源需通过 `assetManager.utils` 转换 URL
- 插屏/压缩纹理/自动图集等资源处理会影响包体大小
- v3.8 起支持仅构建 Bundle，可单独发布选中 Bundle
