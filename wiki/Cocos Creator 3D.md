# Cocos Creator 3D

> [!abstract] 摘要
> Cocos Creator 3D 是 Cocos 引擎在 3D 方向上的"探路者"。2017 年立项、2019 年底以独立产品身份小规模测试、2020 年随 3.0 发布被完全合并。它的存在只有约一年半，但它是 Cocos Creator 3.x 的技术摇篮——现代引擎的 3D 渲染管线、PBR 材质、粒子系统、物理引擎等核心系统都在此分支上首次验证。理解 Creator 3D 的价值不在于"它能做什么"，而在于认识到：没有这个独立分支的验证期，3.0 就不可能在发布时就具备工业级的 3D 游戏开发能力。

## 根本问题：为什么要独立一个 3D 分支？

2017 年，Cocos 团队面临一个艰难的战略选择：

1. **直接在 Creator 2.x 上添加 3D 功能**——这被证明不可行，因为 Cocos2d-x 的图形架构从根上就是为 2D 设计的，加入 3D 意味着推到重来
2. **分叉一个 3D 分支独立开发**——这会暂时分裂产品线，但允许新架构无历史包袱地演进
3. **同时维护两条线**——保持 2D 产品线的稳定运营，同时 3D 分支作为技术验证快速迭代

团队选择了方案 3。这是典型的"并行进化"策略：

- **Creator 2.x** 继续服务 2D 手游市场，保持稳定，不引入破坏性变更
- **Creator 3D** 作为独立产品，从头构建全新的 3D 渲染内核、材质系统、物理引擎，以真实商业项目（如《星空大决战》《最强魔斗士》）进行压力测试

这一策略的核心价值在于**零风险验证**：如果 3D 分支在市场上的表现不理想，2.x 产品线不会受到任何影响；如果 3D 分支成功，则可以在适当时机将两者合并。

## 技术遗产：Creator 3D 为 3.0 贡献了什么

Cocos Creator 3D 并非简单的"添加了 3D 视图的 Creator"。它是一个全新的引擎内核，这些系统后来全部进入了 3.0：

| 系统 | Creator 3D 中的验证 | 进入 3.0 后的状态 |
|------|-------------------|-------------------|
| 3D 渲染管线 | 首次引入，支持 PBR 材质和平行光/点光源 | 成为 3.x 的默认渲染管线 |
| 材质系统 | 首次提出可编程 shader 概念 | 进化为 Effect Asset 系统 |
| 3D 物理 | 首次集成 Bullet 物理引擎 | 成为 3.x 的标准 3D 物理后端 |
| 模型导入 | 支持 glTF/FBX 模型导入和骨骼动画 | 直接继承到 3.0 |
| 粒子系统 | 首次引入 3D 粒子 | 成为 3.x 的标准粒子系统 |
| 编辑器 3D 视图 | 首次在编辑器中实现 3D 场景视口 | 成为 3.x 的基础编辑器体验 |

换句话说，**Creator 3D 是 3.0 的"alpha/beta"阶段**——只不过它以独立产品名发布，让真实的商业项目来验证技术路线。

## 版本时间线

| 时间 | 事件 |
|------|------|
| 2017 | Cocos Creator 3D 立项，开始设计全新的 3D 内核 |
| 2019 年底 | 以 Cocos Creator 3D v1.0 身份发布，面向中国市场小规模测试 |
| 2020 | 迭代至 v1.2，成功上线《星空大决战》《最强魔斗士》等重度商业项目 |
| 2021 年初 | Cocos Creator 3.0 发布，整合 2D 和 3D 产品线；Creator 3D 不再发布独立版本 |
| 2021+ | 原 Creator 3D v1.2 项目可直接升级到 3.0，产品名成为历史 |

## 迁移说明

原 Cocos Creator 3D v1.2 项目可以直接升级到 Cocos Creator 3.0，但需要注意：

- **API 命名变更**：3.0 去掉了组件类名的 `Component` 后缀（如 `MeshRendererComponent` → `MeshRenderer`）
- **资源加载 API**：与 2.4 对齐，对 `loader` 进行了重构
- **后续更新**：升级到 3.0 后，可以继续跟随 3.x 版本迭代，获得 3.8 的所有新特性

## 文档混淆提醒

> [!warning] 注意命名歧义
> 由于 "Cocos Creator 3D" 这个名称已被 3.0 合并，当前文档和社区中提到 "Cocos Creator 3D" 时，**通常**指的是 [[Cocos Creator 概述|Cocos Creator 3.x]] 的 3D 能力，而非这个已废弃的独立版本。搜索资料时注意区分时间节点：2019-2020 年的 "Cocos Creator 3D" 资料指向的是独立版本，2021 年之后的资料指向的是 3.x 的 3D 功能。
