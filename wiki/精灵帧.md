# 精灵帧

> [!abstract] 摘要
> 精灵帧（SpriteFrame）是 UI 渲染基础图形的容器，管理图像的裁剪和九宫格信息，默认持有一个 Texture2D 引用。

## 概述

SpriteFrame 是图像资源导入后生成的子资源，用于 Sprite 组件的渲染。支持自动/自定义/无裁剪三种模式，可设置九宫格边距。Packable 属性控制是否参与动态合图（自定义 Effect 中使用了 UV 坐标时需关闭）。还可通过 RenderTexture 将 3D 相机内容渲染到 SpriteFrame 上显示在 UI 中。
