# 编辑器界面

> [!abstract] 摘要
> Cocos Creator 编辑器由多个可自由移动、组合的面板构成，以场景编辑器、层级管理器、资源管理器、属性检查器为核心工作区。开发者通过所见即所得的编辑方式搭建场景，结合控制台、偏好设置和项目设置面板完成完善的开发体验。

## 核心概念

Cocos Creator 编辑器是一个**基于面板**的集成开发环境，所有面板均支持拖拽移动、调整大小和层叠组合。编辑器默认提供一套布局，用户可根据需求自定义布局并自动保存。

### 编辑器布局

详见 [编辑器布局](raw/editor/editor-layout/index.md)：

- 面板之间可拖拽边界线调整大小
- 拖拽标签栏可将面板移动到任意位置或与其他面板层叠
- 层叠面板在屏幕空间有限时非常实用，可随时拖出恢复独立显示
- 布局修改自动保存在全局配置文件中

### 核心面板

| 面板 | 功能 | 快捷键 |
|------|------|--------|
| 场景编辑器 | 所见即所得的场景搭建，支持 3D/2D 视图切换 | Ctrl/Cmd + 1 |
| 层级管理器 | 树状展示节点层级，支持拖拽排序、搜索、锁定 | Ctrl/Cmd + 4 |
| 资源管理器 | 管理 `assets` 目录下的所有项目资源，支持导入导出 | Ctrl/Cmd + 2 |
| 属性检查器 | 查看和编辑节点、组件、资源的属性 | Ctrl/Cmd + 3 |
| 控制台 | 输出日志、警告、错误信息 | Ctrl/Cmd + 0 |

### 辅助面板

- **工具栏**：位于编辑器顶部，包含变换工具（移动 W / 旋转 E / 缩放 R / 矩形变换 T）、Gizmo 基准点设置、2D/3D 切换、预览和构建按钮。详见 [工具栏](raw/editor/toolbar/index.md)
- **主菜单**：包含文件、编辑、节点、项目、面板、扩展、开发者和帮助 8 个菜单项，集成大部分功能点。详见 [主菜单](raw/editor/mainMenu/index.md)
- **动画编辑器**：编辑关键帧动画和骨骼动画数据（Ctrl/Cmd + 6）
- **预览面板**：支持编辑器内、浏览器和模拟器三种预览方式

## 关键细节

### 场景编辑器

详见 [场景编辑器](raw/editor/scene/index.md)：

- 支持 **3D 视图**（透视相机）和 **2D 视图**（正交相机）
- 3D 视图操作：Alt + 左键旋转，中键平移，滚轮缩放，右键 + WASD 漫游
- 提供五种变换工具：移动、旋转、缩放、矩形变换（仅 UI）、吸附变换
- 变换吸附支持按步长锁定，按住 Ctrl 临时启用
- 对齐与分布工具（2D 视图独有）：支持多节点的顶/底/左/右对齐和均匀分布
- 场景 Gizmo 位于右上角，可快速切换到 6 个正交方向，支持透视图/正交视图切换
- 提供着色模式调试（渲染调试）、场景灯光开关、参考图叠加等辅助功能

### 层级管理器

详见 [层级管理器](raw/editor/hierarchy/index.md)：

- 以树状结构展示场景中的所有节点，与场景编辑器内容同步
- 支持创建空节点、3D 对象、2D 对象、UI 控件、光源、摄像机、特效等
- 搜索功能支持按名称/UUID/Path/组件名称/资源 UUID 检索
- 拖拽节点可改变层级关系或显示顺序
- 节点锁定后无法在场景编辑器中选中（v3.5 起支持级联锁定）
- 支持批量操作：复制粘贴、生成副本、多选（Ctrl/Cmd 或 Shift）

### 资源管理器

详见 [资源管理器](raw/editor/assets/index.md)：

- 管理 `assets` 目录下的项目资源，`internal` 为内置只读资源库
- 支持创建、导入、导出资源包，拖拽资源到场景生成节点
- 搜索支持按名称/UUID/URL/查找 UUID 使用等模式
- 右键菜单提供：复制 UUID/PATH、重新导入、在系统文件管理器中显示等操作
- 特定资源（场景、脚本）双击可进入编辑状态

### 属性检查器

详见 [属性检查器](raw/editor/inspector/index.md)：

- 工具栏支持编辑历史前进后退、锁定面板固定编辑对象
- Node 属性：Position、Rotation、Scale，支持批量修改
- 组件属性：每个挂载的组件展开显示其属性，支持复制/粘贴组件值
- 属性类型分为值类型（数字、字符串、布尔、枚举、颜色）和引用类型（节点、组件、资源）
- 支持通过拖拽脚本或添加组件按钮挂载自定义脚本组件
- Prefab 节点提供取消关联、定位资源、从资源还原、更新到资源等功能

### 偏好设置

详见 [偏好设置](raw/editor/preferences/index.md)：

- **通用设置**：编辑器语言、预览 IP/端口、属性步长
- **外部程序**：配置微信开发者工具、Android NDK/SDK、HarmonyOS NDK/SDK、默认脚本编辑器
- **设备管理器**：管理模拟器/浏览器预览的设备分辨率
- **引擎管理器**：自定义 TypeScript 引擎和原生引擎路径
- **资源数据库**：日志等级、忽略文件 Glob、自动刷新、默认导入类型配置
- **预览**：自动刷新预览、模拟器缓清除、HTTPS 配置
- **构建发布**：缓存策略、纹理压缩缓存
- **实验室**：启用延迟渲染、移动脚本自动修改导入路径、场景即时缓存等实验性功能

### 项目设置

详见 [项目设置](raw/editor/project/index.md)：

- **项目数据**：默认 Canvas 设计分辨率、渲染管线（前向/延迟）、高画质模式
- **Layers**：自定义 0-19 个用户 Layer，配合 Camera Visibility 控制可见性
- **排序图层**：自定义 2D UI 和 3D 物体渲染排序
- **物理**：配置 2D/3D 物理参数
- **脚本**：类字段语义、宽松模式、Import Map、插件脚本排序
- **Macro Config**：引擎宏配置（抗锯齿、多点触摸、Label 缓存池等）
- **功能裁剪**：按需裁剪引擎模块以减小包体
- **压缩纹理**：配置各平台的纹理压缩预设

### 控制台

详见 [控制台](raw/editor/console/index.md)：

- 输出 log（灰）、warn（黄）、error（红）三种级别的日志
- 支持过滤、正则搜索、打开日志文件
- 控制台参数可在偏好设置中配置（日期显示、文字大小）
- 支持自定义输出信息格式：跳转链接、显示图片、定位资源/节点、多语言翻译

## 与其他系统的关系

- [[场景与节点系统]]：层级管理器和场景编辑器是操作节点树的图形界面
- [[资源系统]]：资源管理器是资源导入、管理和引用的图形入口
- [[脚本系统]]：属性检查器中编辑脚本组件声明的 `@property` 属性
- [[引擎架构]]：偏好的引擎管理器支持切换自定义引擎路径
- [[发布系统]]：项目设置中的构建配置直接影响跨平台发布行为

## 注意事项

> [!tip] 布局恢复
> 如果面板布局混乱，可以通过主菜单 **File -> 布局 -> 默认布局** 恢复初始布局。

> [!warning] 场景编辑器不可关闭
> 场景编辑器面板不支持弹出/关闭操作，始终保持在主窗口中。

> [!tip] Prefab 编辑保存
> 编辑 Prefab 节点属性后，务必点击属性检查器右上角的绿色保存按钮，否则修改不会生效。
